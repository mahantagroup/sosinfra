
**SOS Infrabulls** is a premium real estate platform established in 2019, dedicated to providing sophisticated property solutions in Indore, India's cleanest city. This application serves as a comprehensive portal for residential, commercial, and industrial land investments.

---

##  Key Features

###  Property Portfolio
- **Ongoing & Completed Projects:** Detailed showcases of current developments and successful past completions.
- **Dynamic Property Listings:** Search and filter through diverse property types including residential and commercial lands.
- **Detailed Property Views:** Comprehensive information, high-quality imagery, and inquiry options for every listing.

###  Specialized Portals
- **Admin Dashboard:** Full control over property listings, agent management, and site content.
- **HR Panel:** Streamlined tools for managing workforce, referral codes, and agent approvals.
- **Agent Portal:** A dedicated space for agents to track their referrals, team performance, and earnings.

###  Experience & Interface
- **Premium Aesthetic:** Modern, minimalist design with glassmorphism effects and high-impact typography.
- **Smooth Animations:** Utilizes **AOS (Animate on Scroll)** for a refined, fluid user journey.
- **Interactive Elements:** Features custom cursors, high-performance sliders, and floating contact widgets (WhatsApp integration).
- **Responsive Design:** Fully optimized for desktop, tablet, and mobile viewing.

---

##  Technology Stack

- **Frontend:** React.js, Vite
- **Styling:** Vanilla CSS (Advanced properties), FontAwesome
- **Navigation:** React Router DOM
- **Animations:** AOS (Animate On Scroll)
- **State Management:** React Hooks (useState, useEffect)
- **Features:** WhatsApp Business Integration, Custom Cursor, Preloaders

##  Getting Started

### Prerequisites
- Node.js (Latest LTS recommended)
- npm or yarn

### Installation
1. Clone the repository
   ```bash
   git clone [repository-url]
   ```
2. Navigate to the project directory
   ```bash
   cd mahanta
   ```
3. Install dependencies
   ```bash
   npm install
   ```
4. Start the development server
   ```bash
   npm run dev
   ```
